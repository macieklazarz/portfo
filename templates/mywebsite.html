About website

This website is intended to introduce my skills which I use in my daily tasks as well as skills I want to develop to improve my IT knowledge. As my main tasks are related to ETL tools and I want to merge it with Python tools development in the section 'My tools' I post some tools developed by my using Python. Example libraries used to develop my tools are Pandas and Numpy.

The website has been prapaired using templates which are listed in section 'Usefull tools'. I've customized and adjusted one of template the way it could be usefull for my purposes. The website is based on Flask framework. This framework let me to apply some solutions which I wanted to be present on the website. For example in the section 'My tools' you can download tools prepaired by me. Section 'Contact' consist contact form. Data entered in the form is stored in file on the server. In the future I want to modify it the way it will be stored in database. Moreover as it is contact form the data is sent to my email. Next functionality is text message notification which I receive on my cell phone everytime when you submit the form. Libraries exploited to serve mail and notification functionalities are email and twilio.
